# University of Eastern Finland Thesis Template (LaTeX)

A LaTeX version of the university thesis template converted from the official Word template for those who are comfortable writing in LaTeX.

> [!IMPORTANT]
> This repository is **NOT** an official university template.
> The LaTeX files were manually converted from the official Word template and may not fully reflect future updates or formatting requirements.

Users are strongly encouraged to:

- Refer to and follow the latest official university Word template: https://studentuef.sharepoint.com/:w:/s/UEF--Kamu/EQoENzzjAmBPuEiY5-O09AkBUGzFUF3Ocp0yb6o0NzQP7g?e=xh68cL
- Modify the LaTeX source files as needed to meet submission requirements

Last update: **May 2026**

---

## Features

- Thesis structure in LaTeX
- Preconfigured formatting based on the Word template
- Bibliography support
- Modular chapter organization
- Suitable for Overleaf and local LaTeX environments
