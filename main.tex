The law about providing digital services obliges universities, among others, to make sure that their web services, i.e. websites and mobile applications conform to the accessibility criteria. This requirement also applies to published theses.

In Finland, the fulfilment of the accessibility criteria is supervised by The Regional State Administrative Agency, whose website offers additional information (Finnish pages). 

The University of Eastern Finland offers accessible Word templates, for which there are instruction in UEF Intra. Log in with UEF credentials. These instructions are based on those general instructions.

Produce an accessible Word file and turn it into an accessible PDF file. For more information, see the linked SWAY presentations. The instructions in question are meant for UEFians and require UEF credentials to be viewed.

\chapter{Thesis structure}
The structure of the master’s thesis usually follows this pattern:
\begin{itemize}
    \item cover, title page
    \item abstract
    \item foreword
    \item contents
    \item abbreviations
    \item the actual text body of the thesis (introduction, material and methods, results, and discussion)
    \item references, bibliography
    \item appendices (when needed)

\end{itemize}

\chapter{Further instructions for the production of accessible documents}

\section{Document settings}
The size of the document is A4. The page margins are set so that the left border is 2 cm, while the right border is 1.5 cm.

\section{Divide the text into headings and body text}
The appearances of body text and headings have already been set in the template. In the document itself, choose which styles will be used in different parts of the text.

\subsection{Body text}
The font used in the template is Open Sans. You may also use other serif fonts (e.g. Arial, Calibri, Verdana, but they need to be updated in the style). The point size of the body text should be 12 points. 11 points is enough for Open Sans, as it is slightly larger. Line Spacing should be 1.5.

The text is aligned along the left margin. 

Paragraphs are separated with an additional line break.

\textbf{Bold} is used to emphasise text. Italicization or CAPITALS are used only in special cases.

\subsection{Heading levels}
Headings use the same font as body text. The size of Heading 1 is 14 points, and it is in bold. Heading 1 begins from its own page (define in the previous).

The size of Heading 2 is 12 points, turquoise (RBG 7,125,158 and \#077D9E.) The size of Heading 3 is 11 points, and it is written in bold. Each heading is followed by an empty space.

Headings should be used starting from Heading 1. Subheadings may be included in the chapters if there are at least two chapters. Do not use a fourth subheading level.

\section{Use lists}
Unordered list:
\begin{itemize}
    \item subject 1
    \item subject 2
    \item subject 3
\end{itemize}

Ordered list with Arabic numerals:
\begin{enumerate}
    \item subject 1
    \item subject 2
    \item subject 3
\end{enumerate}

Avoid using lists inside each other.
